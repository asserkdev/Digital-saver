## How to Install

### 1. Download APK
- Go to **Releases** tab (above)
- Download `app-release.apk`

### 2. Enable Installation from Unknown Sources
- Go to **Settings** → **Security**
- Enable **Unknown Sources** or **Install unknown apps**

### 3. Install
- Open the downloaded APK file
- Tap **Install**
- Open **Digital Saver**

### 4. Pair with Watch
- Turn on your DIY smartwatch
- Open the app
- Tap the connection bar
- Select your watch

## Hardware Needed

Build your own smartwatch with ~1,100 EGP:
- ESP32 microcontroller
- MAX30102 PPG sensor
- MPU6050 accelerometer
- 1.3" OLED display
- 500mAh battery

See the full project at: https://github.com/Cambric-software/Digital-saver
