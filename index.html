🧠 Advanced Algorithms

HRV Analysis

  • RMSSD (Root Mean Square of Successive Differences)
  • SDNN (Standard Deviation of NN intervals)
  • pNN50 (Percentage of successive RR intervals > 50ms)
  • Poincaré Plot Analysis (SD1, SD2)
  • Autocorrelation-based rhythm stability

AFib Detection Algorithm

  • RR Interval Irregularity (Coefficient of Variation)
  • Successive RR Difference Variation Analysis
  • Rhythm Autocorrelation Measurement
  • Missing Beats Pattern Analysis
  • Irregular Cluster Detection (ML-style scoring)

Vascular Health Analysis

  • Vascular Age Estimation from Pulse Pressure + AI
  • Arterial Stiffness from PWV + Augmentation Index
  • Central (Aortic) Pressure Calculation
  • Subendocardial Viability Ratio (SEVR)
  • Wave Reflection Magnitude Analysis
